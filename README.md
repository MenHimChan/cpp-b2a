# cpp-b2a Learning Project

## 📁 项目结构

```
cpp-b2a/
├── .vscode/
│   ├── tasks.json       # 编译任务配置
│   ├── launch.json      # 调试配置
│   └── keybindings.json # 快捷键绑定
├── part1/               # 第一章内容
│   └── example1.cpp
└── README.md
```

## 🚀 使用方法

### 编译 (Ctrl + Shift + B)
1. 打开 `part1/` 中的任意 `.cpp` 文件
2. 按 **Ctrl+Shift+B** 编译
3. 可执行文件生成在同目录

### 调试 (F5)
1. 打开 `part1/` 中要调试的 `.cpp` 文件
2. 按 **F5** 启动 GDB 调试

## 💡 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+B` | 编译当前文件 |
| `F5` | 调试当前文件 |
| `Ctrl+Shift+P` | 打开命令面板 |

## 📝 示例

已在 `part1/example1.cpp` 中提供示例代码，可直接编译运行。
